Warm-up 2: Simple Calculator

Build a calculator to add or subtract two numbers:


def calculator():
    num1 = float(input("Enter first number: "))
    num2 = float(input("Enter second number: "))
    operation = input("Enter operation (+ or -): ")
    
    if operation == '+':
        result = num1 + num2
    elif operation == '-':
        result = num1 - num2
    else:
        return "Invalid operation"
    
    return f"The result is: {result}"

print(calculator())

üéÆ Rock, Paper, Scissors Game

Run the code below to play against the computer:


import random

def get_computer_choice():
    """Returns the computer's randomly chosen move"""
    choices = ['rock', 'paper', 'scissors']
    return random.choice(choices)

def get_user_choice():
    """Returns the user's move"""
    while True:
        user_choice = input("Enter rock, paper, or scissors: ")
        if user_choice in ['rock', 'paper', 'scissors']:
            return user_choice
        else:
            print("Invalid input! Try again.")

def determine_winner(user_choice, computer_choice):
    """Returns the winner"""
    if user_choice == computer_choice:
        return "Tie!"
    elif (user_choice == 'rock' and computer_choice == 'scissors') or \
         (user_choice == 'paper' and computer_choice == 'rock') or \
         (user_choice == 'scissors' and computer_choice == 'paper'):
        return "You win!"
    else:
        return "Computer wins!"

def play_game():
    """Plays the game"""
    computer_choice = get_computer_choice()
    user_choice = get_user_choice()
    print("Computer chose:", computer_choice)
    print(determine_winner(user_choice, computer_choice))

play_game()

What is AI?

Artificial Intelligence (AI) is about machines that can simulate human intelligence. AI systems can:

  • Reason: draw logical conclusions
  • Learn: from data and improve
  • Problem-solve: find solutions to complex tasks

Examples of AI

  • ChatGPT (Reason)
  • Netflix (Learn)
  • Tesla Autopilot (Problem-solve)

What's the difference between AI, ML, and DL?

  • AI: Machines simulating human intelligence
  • ML (Machine Learning): AI that learns from data
  • DL (Deep Learning): ML using neural networks to learn from raw data
